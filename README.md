# 헌마공신 김만일 기념사업회 홈페이지

조선 최대의 말 목장을 운영하며 국난 극복에 기여한 헌마공신 김만일(1550~1632) 공의 뜻을 기리는 기념사업회 공식 홈페이지입니다.

## 📁 폴더 구조

```

├── index.html          # 메인 HTML (텍스트 수정)
├── css/

├── js/

```

## ✏️ 텍스트 수정 방법

`index.html` 을 메모장·VSCode 등으로 열어 **`★★★ 수정 위치 ★★★`** 주석을 찾아 수정합니다.

| 수정할 내용 | 검색어 |

| 히어로 소개글 | `히어로 소개글 수정 위치` |
| 김만일 생애 본문 | `생애 본문 수정 위치` |
| 연보 | `연보 수정 위치` |
| 사업 개요 | `사업 내용 수정 위치` |
| 통계 숫자 | `통계 수정 위치` |
| 전화·이메일·주소 | `연락처 수정 위치` |
| 교통편 안내 | `교통 안내 수정 위치` |
| 지도 iframe | `여기에 지도 iframe을 붙여넣으세요` |

## 🗺️ 지도 넣기

1. 카카오맵( https://map.kakao.com ) 에서 "서귀포시 서성로 919" 검색
2. 상세 페이지 → **공유** → **지도퍼가기** → iframe 코드 복사
3. `index.html` 에서 `<div class="map-placeholder">...</div>` 블록을 삭제
4. 그 자리에 복사한 iframe 코드를 붙여넣기

## 🎨 색상 변경

`css/style.css` 상단의 `:root` 부분에서 색상 변수를 수정하면 전체 사이트에 반영됩니다.

```css
:root {
  --ink:   #1a1c20;   /* 글자색 (먹) */
  --paper: #f4ecdc;   /* 배경색 (한지) */
  --gold:  #b08542;   /* 포인트 (금박) */
  --dan:   #8b2e1f;   /* 강조 (단청 적) */
}
```
